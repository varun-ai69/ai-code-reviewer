# Security Policy

## Supported Versions

Only the latest version of RepoSage is supported.

## Reporting a Vulnerability

We take security seriously. If you find a security vulnerability, please do not disclose it publicly or file a GitHub issue. Instead, report it directly to the repository maintainers:

*   **Email**: bhoompallykalyanreddy@gmail.com

Please include a detailed description of the threat, including steps to reproduce, the environment, and a proof-of-concept if available.

We will review your submission and respond within 48 hours to coordinate a security patch before public disclosure.
